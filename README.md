Me enfoco en construir sistemas backend robustos y escalables, aplicando buenas prácticas de arquitectura de software, diseño de bases de datos relacionales y desarrollo de APIs siguiendo principios de Clean Architecture. Mi trabajo combina lógica de negocio compleja con persistencia de datos eficiente, tanto en Java/Spring Boot como en PHP/CodeIgniter 4.

## Proyectos Destacados

### [MGMSystem](https://github.com/Denys392/project-cost-management-api)
Sistema backend para la gestión de costos y presupuestos de proyectos mediante estructuras EDT, plantillas reutilizables y análisis de costos históricos.

`Java` `Spring Boot` `JPA` `MySQL` `Maven`

Implementa lógica de negocio compleja para cálculos de estimación y persistencia de datos relacionales.

---

### [ERP-LIGERO-API](https://github.com/Denys392/erp-ligero-api)
API REST modular para la administración de procesos de negocio esenciales: control de usuarios, clientes, inventario de productos y flujo de ventas.

`Java` `Spring Boot` `MySQL` `JPA` `Maven`

Diseño orientado a arquitectura modular, segregación de responsabilidades y buenas prácticas de endpoints.

---

### [CATALOGO-WEB](https://github.com/Denys392/catalogo-web)
Aplicación web Full Stack para la gestión y visualización dinámica de un catálogo de prendas, con búsqueda avanzada, filtrado por categorías y galería.

`PHP` `CodeIgniter 4` `MySQL` `JavaScript` `HTML5` `CSS3`
